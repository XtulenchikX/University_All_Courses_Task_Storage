Описание: В данной лабораторной работе был реализован пакет "Tulenchik_package" на основе калькулятора, который был реализован в 3 семестре.  

__*Состав пакета (модули)*:__
- module_calculate.py (содержит основную функцию calculate, которая вызывается для проведения вычислений);
- module_stdDev.py (содержит функцию std_dev, к которой обращается calculate, если выбрано действие std_dev(среднеквадратичное отклонение));
- module_loadParams.py (содержит функцию read_params, которая получает значение Precision (точность) из файла конфигурации *params.ini* в формате строки);
- module_convPres.py (содержит функцию convert_precision, которая берёт значение precision из PARAMS, а затем преобразует его в целое число, которое является количеством знаков после запятой в результате);
- module_calcprint.py (содержит функци calc_print, с помощью который производится вывод результата работы калькулятора в виде таблицы).

__*Тесты:*__
- test_calculate.py (файл с тестами для функции calculate)  
  Вызов: python -m pytest package/test_calculate.py
- test_convPres.py (файл с тестами для функции convert_precision)  
  Вызов: python -m pytest package/test_convPres.py
- test_stdDev.py (файл с тестами для функции std_dev)  
  Вызов: python -m pytest package/test_stdDev.py
- Вызов всех тестов сразу: python -m pytest package/

*__Информация о калькуляторе:__*

'+' : сложение

'-' : вычитание

'*' : умножение

'/' : деление

'//' : целочисленное деление

'%' : остаток от деления

'std_dev' : стандартное отклонение
